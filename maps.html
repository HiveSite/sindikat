Poster Map Generator

Generate minimalist city map posters

Single-file app using MapLibre, OpenStreetMap and browser export. Enter a city, tweak the layout, then export a PNG.

Free stack: MapLibre GL JS + OpenStreetMap vector style from MapTiler demo source. For production traffic, replace the style URL with your own tile/style endpoint.
